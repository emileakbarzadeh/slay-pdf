Open source Acrobat alternative

An open source Adobe Acrobat alternative for local jobs.

Use Slay PDF when the job is a lightweight PDF edit and you want source-available, browser-local tooling instead of an account-backed document platform.

Transparent app

Slay PDF is AGPL software with source code available on GitHub.

Acrobat-style tasks

Merge, split, sign, rotate, crop, resize, redact, OCR, compress and password protect PDF exports.

Local by default

Documents and edits stay in this browser, and export passwords are never saved.

For common edits, not enterprise document management.

Acrobat is a full commercial document platform. Slay PDF is a focused open source alternative for everyday local PDF edits that do not need that platform.

It is especially useful when you want a quick browser workflow with no Slay PDF account, no subscription and no document upload to a Slay PDF app server.

Open source Acrobat alternative FAQ

Is Slay PDF an Acrobat clone?

No. Slay PDF is an independent open source PDF editor for common local browser editing workflows.

When should I still use Acrobat?

Use Acrobat for enterprise document management, organization-controlled signing and advanced Adobe platform workflows.
