PDF editor software

Use PDF editor software that runs locally in the browser.

Slay PDF is open-source PDF editor software delivered as a static web app for common local document workflows.

Open source

Source code is available from the Slay PDF GitHub repository.

Browser-based

Use a modern browser instead of a heavy desktop suite for common edits.

Private by default

Documents and edits stay in this browser for common workflows.

Software for the everyday PDF jobs.

Traditional PDF editor software can be overkill when the job is merging, splitting, signing, redacting or exporting pages.

Slay PDF keeps that smaller workflow focused and transparent, with static hosting and local browser processing.

PDF editor software FAQ

Is Slay PDF software or a website?

It is a static web app: software that opens from the web and runs common workflows in your browser.

Is it open source?

Yes. The source repository is linked from the app and press kit.

Can it replace enterprise PDF software?

It replaces many common local editing tasks, not every enterprise PDF platform feature.
