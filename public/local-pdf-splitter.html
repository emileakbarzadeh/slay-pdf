Split a PDF without sending it away.

Use Slay PDF when a PDF contains more pages than you need or when a large scan should become several smaller files. The workspace shows every page so you can choose the exact split points before download.

For simple jobs, select the pages you need and export selected pages. For larger bundles, add split markers between sections and download the separated PDFs.

Local PDF splitter FAQ

Can Slay PDF split a PDF into multiple files?

Yes. Add split markers in the workspace and use the download menu to export separate PDFs.

Can I extract specific pages?

Yes. Select the pages you want and export selected pages.

Does splitting upload my PDF?

No. The split workflow runs locally in this browser.

Open Slay PDF, choose the pages or split markers you need and download the separated files.
