A local way to combine PDF pages.

Slay PDF works well when you need one finished PDF from several private files and do not want to upload the originals to a random merge site. Import each PDF, review the page order, delete anything unnecessary and export the combined result.

The app is especially useful for document bundles such as invoices, receipts, school forms, lease packets and scanned paperwork where page order matters.

Local PDF merger FAQ

Can I merge PDFs without uploading them?

Yes. Slay PDF merges and exports PDFs in the browser so source files are not uploaded to an application server.

Can I reorder pages before merging?

Yes. Drag pages into the order you want before exporting the merged PDF.

Can I export only part of the merged PDF?

Yes. Select the pages you want and use export selected pages, or place split markers for multiple output files.
