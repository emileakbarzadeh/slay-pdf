FAQ

Answers for local PDF editing.

Quick answers about Slay PDF, privacy, browser storage, Acrobat-style workflows and the PDF tools that run locally.

Is Slay PDF free?

Yes. Slay PDF is a free local PDF editor for common work such as merging PDFs, splitting PDFs, deleting pages, signing documents, editing pages and exporting a finished file.

Do my PDFs get uploaded?

No. Slay PDF is a static client-side web app. Your PDF documents and edits stay in this browser instead of being uploaded to an application server.

What can it replace Adobe Acrobat for?

Slay PDF covers everyday Acrobat-style tasks: merge, split, reorder, delete pages, rotate, sign, annotate, crop, resize, posterise, redact, OCR English PDFs, export images, extract text, add page numbers and password protect exports.

Does Slay PDF save passwords?

No. Passwords are used for the current encrypted export only. Passwords are never saved.

What are the limits?

Browser memory, CPU, storage quota and WebAssembly support set the real limits. Very large files, heavy OCR jobs and complex scans may still need a desktop PDF workflow.

Slay PDF FAQ

What is Slay PDF?

Slay PDF is a free local PDF editor for splitting, merging, posterising, signing, resizing and editing PDFs in the browser.

Does Slay PDF upload PDF files?

No. Slay PDF is a static client-side web app. Documents and edits stay in the browser instead of being uploaded to an application server.

Is Slay PDF a free Adobe Acrobat alternative?

Yes for everyday PDF jobs such as merge, split, delete pages, reorder pages, sign, annotate, crop, resize, posterise, OCR, export images and protect exported PDFs.

Can Slay PDF redact PDFs?

Slay PDF can add redaction overlays and rasterize redacted pages on export so covered page content is not left as selectable text behind the overlay.

Does Slay PDF save passwords?

No. Passwords are used for the current export and are never saved.
