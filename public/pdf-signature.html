PDF signature FAQ

Can I add a signature to a PDF?

Yes. Add a visual signature mark and place it on the page before export.

Is this certificate signing?

No. Use a dedicated certificate-signing workflow for regulated or organization-controlled signatures.

Can I export only the signed pages?

Yes. Select the pages you need and export selected pages.
