A PDF24 alternative for browser-local PDF tools.

Slay PDF collects common PDF editing, conversion, protection and export tools in one local web app.

Toolbox workflow

Merge, split, compress, OCR, protect, rotate, crop, resize and convert PDF pages from one workspace.

Image support

Import JPG or PNG files, convert images to PDFs and export selected PDF pages as page images.

Local output

Choose the export format and download the result directly from the browser when the job is finished.

One local workspace for common PDF jobs.

If you searched for a PDF24 alternative, Slay PDF is built for the quick utility jobs that happen around real documents: combine files, extract pages, protect an export, clean scans and convert images.

It runs as a static client-side app, so the core editing flow does not rely on uploading documents to Slay PDF.

PDF24 alternative FAQ

Can Slay PDF convert images to PDF?

Yes. Import JPG or PNG files and export them as PDF pages from the browser workspace.

Can Slay PDF password protect a PDF?

Yes. You can encrypt exported PDFs with a password from the export options.
