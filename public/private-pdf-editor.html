Private PDF editor

Edit PDFs without handing them to a random website.

Slay PDF processes documents in your browser. Merge, split, sign, crop, resize, redact and export PDFs while keeping files on your device.

Client-side app

The editor is a static web app; document processing happens in the browser.

Local workspace

Recent workspaces are saved in IndexedDB so you can continue local edits.

Password handling

Passwords are used for export operations and are never saved by the app.

Built for sensitive everyday documents.

PDFs often contain contracts, scans, receipts, IDs, internal notes or client files. Slay PDF is designed so common edits do not require uploading that data to an unknown server.

Browser storage, installed extensions and your own device security still matter. For high-risk documents, use a trusted machine and clear the local workspace when done.
