A browser-based editor for repeat PDF cleanup.

Browser-based editing is useful for quick jobs: rotate a scan, remove a blank page, sign a form, combine files or export selected pages.

Slay PDF keeps those workflows close to the document and avoids turning every small edit into an upload process.

Browser-based PDF editor FAQ

Which browsers work best?

Use a modern browser with WebAssembly and IndexedDB support.

Can I clear recent work?

Yes. Recent workspace data can be cleared from the app.

Does it work without a login?

Yes. Common workflows do not require an account.
