Open source

An open source PDF editor that runs locally.

Slay PDF is an AGPL browser app for practical PDF editing without sending documents through a hosted document processor.

AGPL source

The Slay PDF source code is public on GitHub under the AGPL-3.0 license.

Browser-local work

PDF documents and edits stay in this browser while you merge, split, sign, redact, OCR and export.

No account gate

Open the editor, import documents and export PDFs without creating a Slay PDF account.

For transparent PDF utility work.

Slay PDF is built for the everyday jobs people often reach for a PDF utility to solve: combine files, remove pages, rotate scans, add visible signatures, redact areas and export a cleaned copy.

Because the app is open source and client-side, teams can inspect how the editor works instead of trusting a random upload site with private documents.

Open source PDF editor FAQ

Is Slay PDF open source?

Yes. Slay PDF is published on GitHub under the AGPL-3.0 license.

Does open source mean my PDFs are public?

No. Open source means the app code is public. Your PDF documents and edits stay in your browser workspace.
