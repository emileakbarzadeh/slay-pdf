Separate PDF pages FAQ

Can Slay PDF download multiple PDFs?

Yes. Add split markers and use the separate-pages export to download multiple PDFs.

Can I separate only selected pages?

Yes. Use page selection when only part of the workspace should be exported.

Are separated files created on a server?

No. Common split and export workflows run locally in this browser.
