A practical PDF editor for everyday fixes.

Slay PDF is built for the small PDF jobs that should not require a subscription, account or file upload. Use it for forms, receipts, contracts, scans, client files, study notes and other documents that only need quick edits.

The app runs as a static client-side web app. Your documents and edits stay in this browser, and passwords are never saved.

Edit PDF FAQ

Can I edit a PDF without uploading it?

Yes. Slay PDF runs PDF editing work in the browser so the document is not uploaded to a Slay PDF application server.

What PDF edits can I make?

You can reorder pages, delete pages, rotate, crop, resize, sign, annotate, redact, split, merge, export selected pages, export images and extract text.

Is Slay PDF a full Adobe Acrobat replacement?

No. It focuses on fast local browser workflows for common edits rather than every enterprise Acrobat feature.
