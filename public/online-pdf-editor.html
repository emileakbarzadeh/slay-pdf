Online PDF editor

Edit PDFs online, but keep the work local.

Slay PDF opens in the browser like an online tool, then handles everyday PDF edits locally: merge, split, sign, reorder, redact, resize and export.

Quick start

Open the editor, drop in PDFs or images, make the edit and download the result.

Local processing

PDF work runs in the browser instead of sending files to an app-server conversion queue.

Practical tools

Organize pages, edit pages, add signatures, fill forms, export text and create searchable OCR PDFs.

The useful part of an online PDF editor without the upload habit.

Most online PDF editors are convenient because they are easy to open. Slay PDF keeps that convenience while avoiding a server-side document processing workflow for common edits.

It is built for fast local jobs: combine files, pull out selected pages, remove pages, rotate scans, add a signature, add a watermark, resize paper, redact a copy or export page images.
