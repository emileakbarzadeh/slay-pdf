## Resource-Page Outreach

These are better handled by personal outreach than anonymous directory submissions:

- University, school, and library pages that recommend PDF tools for students.
- Privacy and security blogs that maintain no-upload tool lists.
- Open-source software newsletters looking for small useful tools.
- Freelancer and small-business resource pages that recommend contract, invoice, receipt, and tax-document tools.
- Internal company wikis where employees need a no-upload PDF workflow for confidential documents.

## Follow-Up Cadence

- Day 0: Submit with one short description, source link, license, and privacy model link.
- Day 7: Check for maintainer feedback or automated link-check failures.
- Day 21: If there is no response, leave one concise follow-up only when the project accepts follow-ups.
- Day 45: Mark as stale and move on unless there is active maintainer discussion.

## Tracking Fields

For a private tracker, record:

- Target URL.
- Submission URL or pull request URL.
- Account used.
- Date submitted.
- Status: submitted, needs changes, accepted, rejected, stale.
- Best live backlink URL if accepted.
- Notes about required copy, screenshots, or category changes.
